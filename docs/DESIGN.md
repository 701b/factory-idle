# Factory Idle - 게임 설계 문서

## 전체 구조 (큰 그림)

### 1단계: 행성 식민지화 (기반 구축) ← 현재 구체화
- 채굴 → 가공 → 조립으로 생산 체인
- 전력 등 인프라 구축
- 연구로 새 기계/레시피 해금 및 효율 업그레이드
- idle 핵심: 자동 생산, 오프라인 수익

### 2단계: 로켓 발사 & 상업 (나중에 구체화)
- 로켓 건조가 큰 마일스톤
- 우주정거장을 통한 제품 판매
- 행성별 고정 수요량, 초과분 안팔림
- 제품별 고정 가격 (가공 단계 깊을수록 비쌈)
- 돈으로 구매 전용 자원 구입 가능

### 3단계: 우주선 & 직접 거래 (나중에 구체화)
- 우주선 제작
- 다른 행성과 직접 거래
- 더 넓은 무역 네트워크

---

## 설계 원칙

- **단순함 우선**: 물류(컨베이어, 밸런서 등) 제거. 기계를 짓고 수량만 관리하면 자동 생산
- **명확한 진행감**: 해금 순서가 선형적, "다음에 뭘 해야 하는지" 항상 알 수 있음
- **idle 적합성**: 접속하지 않아도 자원이 쌓이고, 접속 시 의미 있는 선택만

```
[채굴기] ─── Xenoferrite 원석 ──→ [분쇄기] ──→ [제련기] ──→ Xenoferrite 판재
[채굴기] ─── Technum 원석    ──→ [분쇄기] ──→ [제련기] ──→ Technum 막대
[채굴기] ─── Ignium 원석     ──→ [분쇄기] ──→ [제련기] ──→ 코크스
[수집기] ─── 바이오매스       (분쇄/제련 불필요)

                판재 ──────────→ [조립기] ──→ 철 프레임
        판재 + 막대 ──────────→ [조립기] ──→ 회로기판
        프레임 + 막대 ─────────→ [조립기] ──→ 모터

        프레임 + 회로기판 ─────→ [조립기] ──→ 빨간 과학팩
  모터 + 회로기판 + 판재 ─────→ [조립기] ──→ 초록 과학팩
  과학팩(빨+초) + 모터 + 프레임 → [조립기] ──→ 파란 과학팩
```

- 레벨당: 생산 속도 +10%, 전력 소비 +5%

- 비용: `baseCost * (level * 2)`

### 2-3. 비용 배율 공식

```
n번째 기계 비용 = baseCost * costMultiplier^(n-1)
```

## 3. 전력 시스템

전력은 **저장 불가**. 생산 >= 소비여야 정상 가동.

### 3-1. 발전기

| 발전기 ID | 이름 | 연료 | 발전량 | 비용 | 해금 조건 |
|-----------|------|------|--------|------|----------|
| `biomass_burner` | 바이오매스 버너 | 바이오매스 1/s | 20W | 없음 | 시작 |
| `coke_boiler` | 코크스 보일러 | 코크스 0.5/s | 60W | 판재x30, 프레임x5 | 연구 |
| `turbine_generator` | 터빈 발전기 | 코크스 1/s | 200W | 프레임x20, 모터x10 | 연구 |

### 3-2. 전력 부족 시

```
전력비율 = min(1, totalProduction / totalConsumption)
실제생산속도 = 기본생산속도 * 전력비율
```

## 5. Idle 메카닉

### 5-1. 게임 루프 (틱 간격: 100ms)

```
1. 전력 밸런스 계산
2. 전력비율 산출
3. 각 기계 progress += (delta / duration) * 전력비율 * 보너스배율
4. progress >= 1 이면 입력 확인 → 소비/생산 또는 대기
5. 연구 서버: 과학팩 소비 → 연구 진행
6. UI 갱신 (requestAnimationFrame throttle)
```

### 5-2. 오프라인 수익

- 최대 8시간
- 마지막 접속 시점의 "초당 순생산량" 스냅샷 기반
- 오프라인 효율 50% (연구로 상향 가능)

### 5-3. 자동화 단계

- 게임 시작 5분 이내 완전 자동화 도달 목표
- 수동 클릭은 초반 튜토리얼 느낌만

### 5-4. 프레스티지 (1단계 내)

- 로켓 과학 완료 시 "행성 재개발" 가능
- 모든 자원/기계 초기화, 영구 생산 배율 +25% 누적
